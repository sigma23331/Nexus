# 心运岛勋章等级系统设计（v1.1）

为了让勋章系统具备更强的长期激励作用，将其中部分勋章设计为可升级模式。每个勋章下设多个等级（Lv.1 ~ Lv.4），达成更高条件即可自动升级，提升称号、图标和展示效果。一次性成就勋章保持单级不变。

## 二、数据存储与判断逻辑

### 2.1 数据库字段建议（用户表扩展）

```sql
-- 用于统计各类行为的累计/连续值
user_badge_stats (
  user_id,
  diary_days_total,          -- 累计写日记天数（天数去重）
  answer_book_used_total,    -- 答案之书累计使用次数
  login_continuous_days,     -- 当前连续登录天数
  share_total,               -- 累计分享次数
  easter_egg_total,          -- 累计触发彩蛋次数
  collect_total,             -- 累计收藏数量
  social_interact_total,     -- 广场评论+点赞次数
  double_task_continuous,    -- 双重任务（看运势+写日记）连续天数
  last_login_date,           -- 上次登录日期（用于连续登录计算）
  last_diary_date,           -- 上次写日记日期
  last_double_date           -- 上次完成双重任务日期
)
```

### 2.2 勋章等级判断函数（后端示例，Python）

```python
def get_badge_level(user_stats, badge_key):
    thresholds = {
        "心灵笔触": [3, 7, 15, 30],
        "解惑旅人": [10, 50, 200, 1000],
        "恒心之岛": [7, 30, 100, 365],
        "分享微光": [1, 10, 50, 200],
        "彩蛋猎人": [1, 5, 20, 100],
        "收藏小栈": [3, 15, 50, 200],
        "社交浮萍": [1, 10, 50, 200],
        "满月守护": [7, 30, 100]   # 只有3级
    }
    value = user_stats[badge_key]  # 对应的统计字段
    for i, thresh in enumerate(thresholds[badge_key], start=1):
        if value < thresh:
            return i-1  # 返回当前等级（0级表示未获得）
    return len(thresholds[badge_key])
```

### 2.3 触发升级的时机

| 动作 | 更新统计 | 检查勋章 |
| :--- | :--- | :--- |
| 查看当日运势（每日首次） | `login_continuous_days` 更新；若同时写日记则更新 `double_task_continuous` | 恒心之岛、满月守护、好运连莲（一次性） |
| 记录心情日记 | `diary_days_total` 增加（同一天只计一次） | 心灵笔触 |
| 使用答案之书 | `answer_book_used_total` +1 | 解惑旅人 |
| 分享卡片 | `share_total` +1 | 分享微光 |
| 触发彩蛋 | `easter_egg_total` +1 | 彩蛋猎人 |
| 收藏内容 | `collect_total` +1 | 收藏小栈 |
| 广场评论/点赞 | `social_interact_total` +1 | 社交浮萍 |

> 每次更新后，调用勋章检查函数，若等级变化则向用户推送通知（浏览器提醒或应用内弹窗）。

## 三、前端展示建议

### 3.1 勋章墙页面（"我的" 页面入口）

- 以网格形式展示所有勋章，未获得的勋章显示灰色问号或剪影。
- 可升级勋章显示当前等级及下一等级进度条（例如 7/15 天）。
- 点击勋章卡片可展开详细说明：名称、等级、达成条件、获得时间（如果有）。

### 3.2 获得新勋章/升级时的弹窗

- 全屏半透明浮层，展示勋章图标、名称、新称号、动画特效。
- 提供“前往勋章墙查看”和“继续使用”两个按钮。

### 3.3 进度提示（轻量化）

- 在相关功能操作完成后，底部角落出现短暂提示：“日记书写进度：已记录7天，再写3天即可升级心灵笔触Lv.2✨”。
