# frontend

## 环境要求（需手动确认/配置）

- Node.js >= **18.0.0**

## 配置说明（无需手动确认/配置）

| 依赖 | 版本 |
| ---- | ---- |
| **Vue** | 3.5.32 |
| **Vite** | 5.4.21 |
| **TypeScript** | 5.9.3 |
| **Tailwind CSS** | 3.4.19 |
| Vue Router | 4.6.4 |
| Pinia | 3.0.4 |
| Axios | 1.15.1 |
| ECharts | 6.0.0 |
| html2canvas | 1.4.1 |
| dayjs | 1.11.20 |

## 快速开始

```bash
# 1. 进入前端项目目录
cd frontend

# 2. 安装依赖
npm ci
# 安装后出现两条警告，忽略

# 3. 启动开发服务器
npm run dev
# 该命令在终端上保持运行，修改前端页面可实现热更新

# 4.退出
Ctrl + C -> Y
```
